# ScaleUp
## Project summary
### Multi-platform app that helps businesses find how to price their buisnesses to scale at their current size

### Detailed Description
To help smaller businesses learn where their products or services should be priced I am building an app that takes all information into account
to generate a suggested price. This will include current debt, mortgage or rent, goals (eg. new office space), units sold or jobs done with average price,
and more. The goal is that with each new metric included in this calculation the suggested pricing created will allow buisness owners to scale their buissness 
without have to overprice their goods or services.
